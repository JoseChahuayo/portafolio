Semana 1

Tecnologías de Desarrollo Web

Definiciones

Las tecnologías de desarrollo web engloban una diversidad de recursos esenciales que comprenden herramientas, lenguajes de programación, frameworks y estándares. Estos elementos son empleados de manera integral en el proceso de creación y diseño de aplicaciones, así como en la elaboración de sitios web que se despliegan en el vasto universo de internet. La utilización coordinada de estas tecnologías permite a los desarrolladores estructurar, diseñar y optimizar tanto la funcionalidad como la apariencia de las plataformas digitales, contribuyendo así a la creación de experiencias interactivas y eficientes para los usuarios en línea.

Procedimientos

Las tecnologías de desarrollo web se desglosan en dos componentes fundamentales: el Lado Frontend y el Lado Backend.
En el Lado Frontend, se encuentran los lenguajes de programación principales como HTML, CSS, y JavaScript. Estos desempeñan un papel crucial en la creación de la interfaz de usuario y la presentación visual de las aplicaciones y sitios web. HTML proporciona la estructura básica del contenido, CSS define el estilo y el diseño, mientras que JavaScript añade interactividad y dinamismo a la experiencia del usuario.
En el Lado Backend, nos adentramos en la infraestructura que respalda el funcionamiento del sitio web. Aquí, nos encontramos con servidores web y una variedad de lenguajes de programación, entre los que destacan PHP, Java, Python, entre otros. Estos lenguajes son responsables de gestionar la lógica empresarial, la interacción con bases de datos, y la respuesta a las solicitudes del usuario.

Resultados

Se adquirió la habilidad de distinguir entre diversas tecnologías web que posibilitan a los desarrolladores la creación de aplicaciones web efectivas y funcionales. Este conocimiento abarca desde la concepción de la estructura y el diseño, hasta la implementación de la funcionalidad y la gestión eficiente de datos. En esencia, se ha obtenido una comprensión integral que abarca todo el espectro del desarrollo web, permitiendo a los profesionales abordar cada aspecto crítico, desde la apariencia visual hasta la experiencia del usuario y el manejo robusto de la información.

Semana 2

HTML

Definiciones

HTML, acrónimo de HyperText Markup Language, representa un estándar fundamental para definir la estructura y el formato de una página web. Este lenguaje posibilita la presentación de diversos elementos, como texto, imágenes, enlaces, formularios y otros elementos multimedia, permitiendo su visualización y navegación a través de un navegador web.

Procedimientos

Dentro de las características que ofrece HTML, destacan diversas funcionalidades que contribuyen a la organización efectiva del contenido de una página web. Esto incluye la capacidad de estructurar información mediante encabezados, párrafos, listas, tablas y formularios, facilitando así la comprensión y navegación por parte de los usuarios. Además, HTML posibilita la creación de hipervínculos o enlaces (links) y la implementación de formularios interactivos, permitiendo a los usuarios visitar páginas, proporcionar datos o expresar comentarios.

Resultados

En el proceso de aprendizaje, se ha comprendido que HTML desempeña un papel crucial en la creación de páginas web al definir tanto la estructura como el contenido. Se ha identificado que trabaja de manera complementaria con otros lenguajes como CSS y JavaScript para lograr resultados más avanzados en términos de presentación y funcionalidad. Asimismo, se reconoce la versatilidad de HTML al posibilitar la creación de interfaces interactivas y atractivas en el vasto panorama de la web.

Semana 3

CSS y Librerias

Definiciones

CSS, o Cascading Style Sheets, representa un lenguaje esencial utilizado para gestionar la presentación y el diseño de páginas web. Este enfoque se centra en aspectos visuales cruciales, tales como colores, fuentes, márgenes, tamaños y posiciones de elementos en la página, permitiendo una personalización detallada de la apariencia visual de un sitio.

Procedimientos

Entre sus funciones clave, CSS proporciona la capacidad de establecer un diseño coherente, facilitando la creación de diseños uniformes en todo un sitio web. Además, colabora estrechamente con bibliotecas como Bootstrap, un framework ampliamente utilizado que ofrece una colección de estilos predefinidos y componentes reutilizables, agilizando así la creación rápida de sitios web responsivos. También se integra con herramientas como Less, que aporta funcionalidades avanzadas, incluyendo variables, anidamiento y mixins, simplificando la escritura de código y permitiendo una mayor flexibilidad en el desarrollo.

Resultados

En términos generales, CSS desempeña un papel crucial al separar el contenido de la presentación, lo que facilita la creación de diseños consistentes y atractivos en un sitio web. Este proceso se ve fortalecido por el apoyo de diversas bibliotecas y frameworks CSS, que no solo simplifican el desarrollo, sino que también posibilitan la creación de diseños y componentes web avanzados. En consecuencia, CSS se erige como una herramienta indispensable para lograr experiencias visuales atractivas y funcionales en el entorno digital.

Semana 4

Java Script

Definiciones

JavaScript, un lenguaje de programación de alto nivel, se destaca como un elemento fundamental en el desarrollo web al ser interpretado por los navegadores. Su función primordial radica en dotar a las páginas web de un carácter interactivo y dinámico, permitiendo una experiencia más inmersiva y atractiva para los usuarios.

Procedimientos

Las múltiples facetas de JavaScript en el desarrollo web le confieren una versatilidad excepcional. En el ámbito de la interactividad en el lado del cliente, este lenguaje permite crear respuestas dinámicas a las acciones del usuario, mejorando la usabilidad. Además, desempeña un papel crucial en la comunicación con el servidor, posibilitando el envío y recepción de datos de manera asíncrona mediante tecnologías como XMLHttpRequest y la API Fetch. Asimismo, JavaScript se adentra en la automatización del navegador web, brindando herramientas para realizar acciones programáticas que optimizan procesos y mejoran la eficiencia.

Resultados

En el tejido del desarrollo web contemporáneo, JavaScript emerge como un componente indispensable. Su capacidad para infundir interactividad y dinamismo en las páginas web contribuye directamente a una experiencia de usuario enriquecida. Este lenguaje no solo mejora la funcionalidad, sino que también sirve como piedra angular para la creación de aplicaciones web avanzadas. En resumen, JavaScript no solo cumple un papel técnico, sino que se erige como un habilitador clave para la innovación y la creación de experiencias digitales altamente atractivas y funcionales.

Semana 5

Tecnologías de Desarrollo Web

Definiciones

React, una destacada librería de JavaScript, tiene como propósito principal la creación de interfaces de usuario dinámicas y eficientes. Además, se destaca por su utilidad en el desarrollo de aplicaciones de una sola página (SPA), ofreciendo soluciones modernas y efectivas para la construcción de experiencias interactivas en la web.

Procedimientos

React se despliega en una variedad de propósitos dentro del desarrollo web y móvil. En el ámbito del desarrollo web, posibilita la creación de interfaces interactivas para aplicaciones web, mientras que, a través de su extensión React Native, se utiliza para el desarrollo de aplicaciones móviles, proporcionando una base común para la creación de experiencias consistentes en diversas plataformas. La capacidad de React para integrarse sin problemas con otras bibliotecas y frameworks amplía su versatilidad y flexibilidad en el proceso de desarrollo.

Resultados

En resumen, React se erige como una biblioteca esencial de JavaScript, aprovechada para la construcción de interfaces de usuario dinámicas y eficientes en aplicaciones tanto web como móviles. Su rendimiento optimizado y su capacidad para facilitar el desarrollo rápido se traducen en resultados tangibles, permitiendo a los desarrolladores crear experiencias digitales atractivas y altamente funcionales. Con React, se abre la puerta a la innovación y a la creación eficaz de aplicaciones que cumplen con los estándares de rendimiento y experiencia de usuario en constante evolución.
